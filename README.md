# react-native-usb-printer-manager

lib to manage usb port

## Installation

```sh
npm install react-native-usb-printer-manager
```

## Usage


```js
import { multiply } from 'react-native-usb-printer-manager';

// ...

const result = await multiply(3, 7);
```
